# Thaha - Real-Time Messaging Web Application
From Unknown to Known.

Thha is a modern real-time messaging platform built to explore how ideas, conversations, and information travel instantly through technology.

Inspired by the meaning of **थाहा (Thaha)** — *"to know"* — this project represents the transformation from traditional human communication methods into modern digital communication systems.

---

## 📖 The Story Behind Thha

Before the internet, servers, and fiber-optic networks, communication was carried by people.

In Nepal, philosopher **Rupchandra Bista** traveled across communities in the hills during the 1970s. To spread awareness and challenge ignorance, he wrote a simple but powerful word on public spaces:

**थाहा (Thaha) — "To Know"**

Those messages on rocks and walls acted as an early form of public information sharing — a way to move knowledge from one person to another.

Modern messaging systems follow the same philosophy.

A message is created by one person, transmitted through infrastructure, stored in databases, and delivered instantly to another person.

Thha brings that idea into software:

**Turning traditional public walls into a modern digital conversation space.**

# 🚀 Features

## Real-Time Messaging

- Instant one-to-one communication
- WebSocket-based real-time message delivery
- No polling delay
- Live conversation updates

## Authentication & Security

- JWT-based authentication
- Protected API endpoints
- Secure password encryption using BCrypt
- User authorization handling

## User Management

- User registration
- Login system
- User discovery
- Profile information management

## Conversation System

- Private conversations
- Conversation history
- Participant management
- Persistent message storage

Future vision:

When a recipient opens a message, the read status represents the philosophy of Thha:
The receiver now knows.
